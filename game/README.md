# DOTS & BOXES - Hướng dẫn chạy khi clone từ Git

## Các bước chạy game sau khi clone về

### 1. Cài đặt Python

Đảm bảo máy đã cài Python 3.8 trở lên: https://www.python.org/downloads/

### 2. Mở Terminal tại thư mục `dot-box`

```
cd dot-box
```

### 3. Cài đặt các thư viện phụ thuộc

```
pip install -r requirements.txt
```

### 4. Chạy game

```
python index.py
```

### 5. Lưu ý

- Đảm bảo thư mục `img/` và `music/` có dữ liệu hình ảnh, nhạc nền.
- Nếu gặp lỗi font, hãy cài đặt font Unicode hỗ trợ tiếng Việt trên hệ điều hành.
- Nếu muốn reset điểm cao, xóa file `highscores.json`.
